## How to Read This Changelog

Each skill has its own version history under its name. Versions follow [Semantic Versioning](https://semver.org/):

- `MAJOR`: Incompatible changes
- `MINOR`: New features, backward compatible
- `PATCH`: Bug fixes, backward compatible

Example:
```
### xiaoyu-knowledge-learning
- `1.2.0` - Added interactive learning flow
- `1.1.0` - Improved knowledge tree positioning
- `1.0.1` - Fixed typo in SKILL.md
- `1.0.0` - Initial release
```
