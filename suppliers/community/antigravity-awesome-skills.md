## 安装方式

### Claude Code

```bash
# 克隆仓库
git clone https://github.com/sickn33/antigravity-awesome-skills.git

# 复制需要的 skill 到 Claude Code skills 目录
cp -r antigravity-awesome-skills/skills/code-review ~/.claude/skills/
```

### Cursor

```bash
# 复制到 Cursor skills 目录

```

### OpenClaw

