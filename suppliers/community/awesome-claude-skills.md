## 📦 安装方式

### 方式 1：直接安装（推荐）

```bash
# 克隆仓库
git clone https://github.com/ComposioHQ/awesome-claude-skills.git

# 复制到 Claude Code
cp -r awesome-claude-skills/skills/* ~/.claude/skills/
```

### 方式 2：选择性安装

```bash
# 只安装文档处理 Skills
cp -r awesome-claude-skills/skills/document-skills ~/.claude/skills/

# 只安装开发工具
cp -r awesome-claude-skills/skills/mcp-builder ~/.claude/skills/
```
